# web-reGOn


## [Requirements]
You should install [go](https://go.dev/doc/install)
## [Usage]
```bash
Flags:
  -D, --dns      HTTP Status Code
  -h, --help     help for linux-reGOn
  -H, --http     DNS Records
  -S, --shodan   Shodan Host IP Query (api key needed)
```
## [Input]
For Linux:
```bash
./linux-reGOn [Flag] $URL
```
