# web-reGOn


## [Requirements]
You should install [go](https://go.dev/doc/install)

For Mac:
```bash
brew update && brew install go
```
## [Usage]
```bash
Flags:
  -h:   Prints HTTP Status Code
  -d:  Prints DNS Records
```
## [Input]
For Linux:
```bash
./linux-reGOn [Flag] $URL
```
For Mac:
```bash
./mac-reGOn [Flag] $URL
```
